## 关键决策节点说明

### 1. 初始治疗方案选择依据
- **疾病分度**：主要决策依据
- **生育需求**：影响药物选择
- **合并症**：影响治疗方案
- **患者偏好**：尊重患者选择

### 2. 治疗调整时机
- **定期评估**：每3个月必须评估
- **症状变化**：症状加重立即评估
- **不良反应**：出现严重不良反应需调整

### 3. 疗效评估标准
- **临床症状改善率**：主要评估指标
- **影像学改善**：客观评估指标
- **生活质量提高**：综合评估指标

### 4. 停药决策依据
- **完全缓解持续时间**：至少6个月
- **客观检查结果**：影像学正常
- **患者意愿**：考虑患者需求

## 特殊情况处理流程

### 1. 计划妊娠患者
```
计划妊娠 → 停药3个月 → 尝试妊娠 → 妊娠期管理
```

### 2. 治疗无效患者
```
当前方案无效 → 重新评估分度 → 升级治疗方案 → 考虑手术干预
```

### 3. 不良反应处理
```
出现不良反应 → 评估严重程度 → 调整剂量/停药 → 对症处理 → 重新评估方案
```

## 质量控制要点

1. **规范化评估**：使用标准化评估工具
2. **定期监测**：严格执行监测计划
3. **多学科协作**：必要时请相关科室会诊
4. **患者教育**：确保患者理解治疗方案
5. **记录完整**：详细记录治疗过程和决策依据
