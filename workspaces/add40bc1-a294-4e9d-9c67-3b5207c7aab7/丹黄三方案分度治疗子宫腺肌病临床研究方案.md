# "丹黄三方案"分度治疗子宫腺肌病的理想临床研究方案设计

## 一、研究方案概述

### 1.1 方案名称
"丹黄三方案"分度治疗轻、中、重度子宫腺肌病的多中心、随机、双盲、安慰剂对照临床试验

### 1.2 方案版本
V1.0 (2024年12月)

### 1.3 研究背景
子宫腺肌病（Adenomyosis，AM）是子宫内膜腺体和间质侵入子宫肌层引起的良性病变，临床表现为进行性痛经、月经量增多、经期延长等。丹黄三方案是基于丹黄祛瘀胶囊的中西医结合分度治疗方案，前期研究显示其治疗子宫腺肌病具有显著疗效。

### 1.4 研究依据
- 2024年《"丹黄三方案"分度治疗轻、中、重度子宫腺肌病的临床研究》
- 2023年《子宫腺肌病中西医结合诊疗指南》
- 现有临床研究文献综述（10篇相关研究）

## 二、研究目的和假设

### 2.1 主要研究目的
评价"丹黄三方案"分度治疗子宫腺肌病的临床疗效和安全性。

### 2.2 次要研究目的
1. 比较不同分度（轻、中、重度）患者的疗效差异
2. 评估治疗对患者生活质量的影响
3. 探索丹黄三方案的作用机制

### 2.3 研究假设
**主要假设**：与安慰剂相比，丹黄三方案分度治疗能显著改善子宫腺肌病患者的痛经评分（VAS评分降低≥50%）。

**次要假设**：
1. 丹黄三方案能显著缩小子宫体积（超声测量）
2. 丹黄三方案能显著降低血清CA125水平
3. 重度患者采用丹瑞方案（丹黄祛瘀胶囊+醋酸曲普瑞林）疗效最优

## 三、研究设计类型

### 3.1 设计类型
多中心、随机、双盲、安慰剂对照、平行组设计的优效性临床试验

### 3.2 设计特点
1. **分度分层随机**：根据病情分度（轻、中、重度）进行分层随机
2. **适应性设计**：允许中期分析后调整样本量
3. **长期随访**：治疗期6个月，随访期12个月

### 3.3 研究中心
计划在全国8-10家三级甲等医院妇科开展，包括：
- 北京协和医院
- 复旦大学附属妇产科医院
- 四川大学华西第二医院
- 中山大学附属第一医院等

## 四、研究对象选择

### 4.1 纳入标准
1. 年龄18-45岁，有月经周期的女性
2. 经临床和影像学（经阴道超声或MRI）确诊为子宫腺肌病
3. 痛经VAS评分≥4分（10分制）
4. 符合中医血瘀证诊断标准
5. 自愿参加研究并签署知情同意书

### 4.2 排除标准
1. 妊娠或哺乳期妇女
2. 合并子宫内膜异位症、子宫肌瘤等其他妇科疾病
3. 严重肝肾功能不全（ALT/AST>2倍正常上限，Cr>1.5倍正常上限）
4. 对研究药物成分过敏
5. 近3个月内使用过激素类药物或中药治疗
6. 有恶性肿瘤病史

### 4.3 分度标准（基于MUSA超声评分系统）
**轻度**：MUSA评分1-3分，子宫体受影响<25%
**中度**：MUSA评分4-6分，子宫体受影响25-50%
**重度**：MUSA评分>7分，子宫体受影响>50%

### 4.4 退出标准
1. 出现严重不良事件
2. 依从性差（服药依从性<80%）
3. 患者要求退出
4. 研究者判断需要退出

## 五、干预措施描述

### 5.1 试验分组
根据分度进行分层随机，每组再随机分为试验组和对照组：

| 分度 | 试验组（丹黄三方案） | 对照组（安慰剂+标准治疗） |
|------|-------------------|------------------------|
| 轻度 | 丹黄方案：丹黄祛瘀胶囊 4粒/次，3次/日 | 安慰剂胶囊+基础止痛治疗 |
| 中度 | 丹米方案：丹黄祛瘀胶囊+米非司酮10mg/日 | 安慰剂胶囊+米非司酮10mg/日 |
| 重度 | 丹瑞方案：丹黄祛瘀胶囊+醋酸曲普瑞林3.75mg/月 | 安慰剂胶囊+醋酸曲普瑞林3.75mg/月 |

### 5.2 药物规格
1. **丹黄祛瘀胶囊**：0.4g/粒，国药准字Z20010033
2. **米非司酮片**：10mg/片
3. **醋酸曲普瑞林注射液**：3.75mg/支
4. **安慰剂胶囊**：外观、大小、颜色与丹黄祛瘀胶囊一致

### 5.3 治疗周期
- **治疗期**：6个月
- **随访期**：治疗结束后继续随访6个月
- **总研究时间**：12个月

### 5.4 合并用药规定
允许使用非甾体抗炎药（NSAIDs）控制急性疼痛，但需记录使用情况。

## 六、对照设置

### 6.1 对照类型
安慰剂对照+标准治疗对照

### 6.2 对照合理性
1. **伦理合理性**：所有患者都接受基础治疗，试验组在基础治疗上加用丹黄祛瘀胶囊
2. **科学合理性**：能准确评估丹黄祛瘀胶囊的附加疗效
3. **临床合理性**：符合临床实践，易于实施

### 6.3 盲法设计
**双盲设计**：
1. 患者不知分组情况
2. 研究者不知分组情况
3. 统计分析师不知分组情况

**揭盲条件**：
1. 研究结束时
2. 发生严重不良事件需要紧急救治时
3. 数据安全监查委员会要求时

## 七、主要和次要结局指标

### 7.1 主要结局指标
**痛经视觉模拟评分（VAS）改善率**
- 定义：治疗6个月后VAS评分较基线降低≥50%的患者比例
- 测量时点：基线、治疗3个月、治疗6个月、随访3个月、随访6个月
- 测量方法：10cm VAS标尺，患者自评

### 7.2 次要结局指标

#### 7.2.1 症状相关指标
1. **月经失血图（PBAC）评分变化**：评估月经量变化
2. **痛经持续时间**：每次痛经持续小时数
3. **非经期盆腔痛VAS评分**

#### 7.2.2 影像学指标
1. **子宫体积变化**：经阴道超声测量子宫三径计算体积
2. **结合带厚度（JZmax）**：MRI测量，诊断标准>12mm
3. **病灶范围变化**：MUSA超声评分

#### 7.2.3 实验室指标
1. **血清CA125水平变化**
2. **血液流变学指标**：全血黏度、血浆黏度、红细胞压积
3. **炎症因子**：TNF-α、IL-6、VEGF水平

#### 7.2.4 生活质量指标
1. **子宫腺肌病健康相关生活质量量表（AM-HRQoL）**
2. **中医证候积分**：根据血瘀证候评分
3. **焦虑抑郁评分**：HADS量表

#### 7.2.5 安全性指标
1. **不良事件发生率**
2. **严重不良事件发生率**
3. **实验室检查异常率**
4. **生命体征变化**

## 八、样本量计算

### 8.1 计算依据
基于主要结局指标（VAS评分改善率）进行样本量计算

### 8.2 参数设定
1. **检验水准**：α=0.05（双侧）
2. **检验效能**：1-β=0.90
3. **预期疗效**：
   - 试验组改善率：70%
   - 对照组改善率：50%
4. **脱落率**：预计20%

### 8.3 计算公式
采用两组率比较的样本量公式：
```
n = [(Zα/2 + Zβ)^2 × (p1(1-p1) + p2(1-p2))] / (p1 - p2)^2
```
其中：
- Zα/2 = 1.96 (α=0.05)
- Zβ = 1.28 (β=0.10)
- p1 = 0.70 (试验组预期改善率)
- p2 = 0.50 (对照组预期改善率)

### 8.4 计算结果
1. **每组所需样本量**：n = 156例
2. **考虑脱落率后**：n = 156/(1-0.2) = 195例
3. **按分度分层**：轻、中、重度各需65例×2组=130例
4. **总样本量**：195×3=585例

### 8.5 样本量分配
| 分度 | 试验组 | 对照组 | 小计 |
|------|--------|--------|------|
| 轻度 | 98例 | 98例 | 196例 |
| 中度 | 98例 | 98例 | 196例 |
| 重度 | 98例 | 98例 | 196例 |
| **总计** | **294例** | **294例** | **588例** |

## 九、随机化和盲法设计

### 9.1 随机化方法
**分层区组随机化**：
1. **分层因素**：研究中心、病情分度（轻、中、重度）
2. **区组长度**：4或6的倍数，保持盲态
3. **随机序列生成**：由独立统计学家使用SAS PROC PLAN生成
4. **分配隐藏**：采用中央随机化系统

### 9.2 盲法实施
#### 9.2.1 药物盲法
1. **双模拟技术**：试验组用丹黄祛瘀胶囊+安慰剂片，对照组用安慰剂胶囊+活性药片
2. **外观一致性**：所有胶囊外观、大小、颜色、气味一致
3. **包装一致性**：采用相同包装和标签

#### 9.2.2 人员盲法
1. **患者**：不知治疗分配
2. **研究者**：不知治疗分配
3. **结局评估者**：独立于治疗实施
4. **统计分析师**：分析时保持盲态

#### 9.2.3 盲态维护
1. **紧急揭盲信封**：各中心配备
2. **盲态核查**：定期进行盲态完整性评估
3. **揭盲程序**：研究结束后统一揭盲

## 十一、统计分析计划

### 11.1 分析数据集
1. **全分析集（FAS）**：所有随机化患者，采用意向性治疗（ITT）原则
2. **符合方案集（PPS）**：完成治疗方案且无重大方案违背
3. **安全性分析集（SS）**：至少接受一次治疗的患者

### 11.2 主要结局分析
**主要分析**：比较试验组和对照组VAS评分改善率
- **统计方法**：卡方检验或Fisher精确检验
- **效应量**：相对危险度（RR）及其95%置信区间
- **分层分析**：按病情分度进行亚组分析

### 11.3 次要结局分析
1. **连续变量**：采用t检验或Mann-Whitney U检验
2. **重复测量数据**：采用混合效应模型
3. **生存分析**：采用Kaplan-Meier法和Log-rank检验
4. **多因素分析**：采用多元回归模型调整混杂因素

### 11.4 亚组分析
1. 按病情分度（轻、中、重度）分层分析
2. 按年龄分层（<35岁 vs ≥35岁）
3. 按基线VAS评分分层（<7分 vs ≥7分）

### 11.5 安全性分析
1. **不良事件描述**：发生率、严重程度、因果关系
2. **实验室异常**：临床意义异常值分析
3. **生命体征变化**：描述性统计

### 11.6 缺失数据处理
1. **主要分析**：采用多重插补法处理缺失数据
2. **敏感性分析**：采用最差情况分析、完整病例分析

### 11.7 统计软件
SAS 9.4或R 4.0以上版本

## 十二、伦理考虑

### 12.1 伦理审查
1. **中心伦理委员会**：主要研究者所在机构伦理委员会
2. **各中心伦理审查**：所有参与中心均需获得伦理批准
3. **持续审查**：每年提交进展报告

### 12.2 知情同意
1. **知情同意过程**：充分解释研究目的、风险、获益
2. **知情同意书**：使用患者能理解的语言
3. **知情同意文件**：一式两份，患者和研究者各保留一份

### 12.3 风险获益评估
**潜在风险**：
1. 药物不良反应
2. 检查相关不适
3. 时间成本

**潜在获益**：
1. 可能获得有效治疗
2. 免费检查和随访
3. 为医学研究做贡献

### 12.4 隐私保护
1. **数据匿名化**：使用唯一识别码代替个人信息
2. **数据安全**：加密存储，访问控制
3. **结果发表**：不披露患者身份信息

### 12.5 利益冲突声明
所有研究者需声明与药厂的经济利益关系

## 十五、研究局限性及应对策略

### 15.1 预期局限性
1. **患者脱落**：可能影响结果完整性
2. **盲法破盲**：可能影响结果客观性
