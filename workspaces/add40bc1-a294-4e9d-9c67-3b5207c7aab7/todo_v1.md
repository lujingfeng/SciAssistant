# Task Planning Document
## task_name: 丹黄三方案治疗子宫腺肌病临床研究
## task_desc: 系统研究"丹黄三方案"分度治疗轻、中、重度子宫腺肌病的临床疗效、安全性及作用机制，包括方剂组成、分度标准、临床研究设计、疗效评价指标等
## deliverable_contents: 一份全面的临床研究报告，包括文献综述、研究方法、结果分析、讨论和结论，字数不少于10,000字
## success_criteria: 
1. 全面收集"丹黄三方案"的相关文献资料
2. 明确子宫腺肌病轻、中、重度的分度标准
3. 收集至少10篇相关临床研究文献
4. 分析不同分度下的疗效差异
5. 评估临床研究的证据等级和质量
6. 形成完整的临床研究报告框架
## context: 用户要求研究"丹黄三方案"分度治疗轻、中、重度子宫腺肌病的临床研究，这是一个中医药治疗妇科疾病的临床研究课题
