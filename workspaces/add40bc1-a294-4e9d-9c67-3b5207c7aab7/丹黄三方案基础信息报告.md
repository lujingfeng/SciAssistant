# 丹黄三方案基础信息报告

## 一、方剂概述

**丹黄三方案**是一种现代中医药治疗方案，主要用于子宫腺肌病的分度治疗。该方案基于丹黄祛瘀胶囊，结合分度治疗原则，针对轻、中、重度子宫腺肌病患者制定不同的治疗方案。

## 二、方剂组成

### 主要成分（基于丹黄祛瘀胶囊）
丹黄三方案的核心药物为丹黄祛瘀胶囊，其主要成分包括：

1. **补气药**：黄芪、党参、山药、白术
2. **活血化瘀药**：丹参、当归、鸡血藤、三棱、莪术、土鳖虫、延胡索
3. **清热利湿药**：土茯苓、鱼腥草、败酱草、苦参
4. **温经散寒药**：肉桂、炮姜
5. **通络止痛药**：全蝎、川楝子
6. **固涩药**：芡实

**药材总数**：20味药材

### 具体药材列表
- 黄芪、丹参、党参、山药、土茯苓、当归、鸡血藤、芡实

- 土鳖虫、延胡索、川楝子、苦参

### 主要功效
1. **活血止痛**：改善血液循环，缓解疼痛
2. **软坚散结**：软化硬结，消散肿块
3. **祛瘀解毒**：消除瘀血，清热解毒
4. **益气补虚**：补益气血，增强体质

### 主治病症
1. **主要适应症**：子宫腺肌病（分度治疗）
   - 轻度子宫腺肌病
   - 中度子宫腺肌病  
   - 重度子宫腺肌病
